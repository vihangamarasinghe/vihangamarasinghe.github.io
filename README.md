# Vihanga Marasinghe – Personal Website

Minimal, performance-focused personal site built with Astro and Tailwind CSS.

## Goals

- Fast, static-first pages
- Clear structure and easy maintenance
- SEO-friendly, accessible markup

## Tech Stack

- Astro
- Tailwind CSS
- GitHub Pages + GitHub Actions

## Commands

| Command | Action |
| --- | --- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server |
| `npm run build` | Build production site |
| `npm run preview` | Preview production build |
